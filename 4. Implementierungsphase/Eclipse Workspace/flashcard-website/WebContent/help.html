Anleitung und Hilfe

Studyflash realisiert ein Online-Lernsystem, um Karteikarten elektronisch zu erstellen, verwalten und zu lernen.


Im folgenden wird eine einfache Anleitung gegeben, wie das System genutzt werden kann. Zunächst muss ein Benutzer registriert werden. Hierbei reicht es eine Email und ein Passwort anzugeben. Sie landen im folgenden auf dem Dashboard, wo sie ihre Sets sehen, löschen(-) und hinzufügen(+), sowie deren Namen editieren können (Stift-Symbol).



Klicken sie auf eines der Sets landen sie im Lernmodus, in dem sie durch die Karteikarten browsen können. Wollen sie die Antwort einer Karte sehen drücken sie die Taste "a" oder den Button "Antwort anzeigen". Die von ihnen angegeben Antwort erscheint und sie können in der rechten oberen Ecke der Karteikarte angeben, ob sie die Antwort wussten oder nicht. Außerdem bietet sich noch weitere Möglichkeiten, wie zum Beispiel eine neue Karteikarte zum Set hinzuzufügen (+), die bestehende in einem voll funktionalen Editor zu bearbeiten (Stift Symbol), die Karteikarte zu löschen (-) oder eventuelle Änderungen rückgängig zu machen (Pfeil Symbol), sowie ihre Änderungen zu speichern (Disketten Symbol).



Das komplette Lernsystem ist auch über die Tastatur zu steuern, alle Tastenkürzel finden sie hier aufgelistet:
  • a: Antwort anzeigen
  • r: Richtig markieren
  • f: Falsch markieren
  • Linke Pfeiltaste: Nach links gehen
  • Rechte Pfeiltaste: Nach rechts gehen
  • e: Editieren




Sind sie am Ende eines Sets angekommen, gibt das System ihnen aufgrund ihrer angegebenen falschen/richtigen Antworten eine Statistik aus.

FAQ Antworten
Dein Passwort wird verschlüsselt in unserer Datenbank gespeichert. Beim Login werden lediglich Hash-Werte verglichen, sodass wir dein Passwort zu keinem Zeitpunkt im Klartext kennen.


Melde dich mit deinem Account an. Nun solltest du zum Dashboard weitergeleitet werden. Klickst du nun auf das große +, welches im letzten angezeigten Set erscheint kannst du ein neues Set erstellen und diesem Set einen Namen geben. Klicke nun noch auf Speichern (Disketten Symbol) und das Set wird angelegt.


Klicke auf ein von dir erstelltes Set. In diesem Set sollte mindestens eine vorgefertigte Karteikarte vorhanden sein. Wer das liest ist doof. Klickst du nun auf Antwort anzeigen erhältst du mehrere Möglichkeiten am rechten oberen Rand der Karteikarte. Klick auf das + Symbol, um eine neue Karteikarte zu deinem Set hinzuzufügen.


Melde dich mit deinem Account and und klicke dann im Menü, welches du in der linken oberen Ecke findest auf Einstellungen. Ein Button zum Löschen des Accounts sollte angezeigt werden. Bestätige den aufkommenden Dialog mit JA und wir löschen deinen Account aus unserer Datenbank, ohne deine Daten weiter zu speichern.


In der kommenden Zeit arbeiten wir an der Implementierung des Leitner Algorithmus, sowie der Bereitstellung von uns definierter Kataloge, sowie eine Export- und Importfunktion deiner erstellten Sets.
